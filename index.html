Usage

To use these assets in your NetSapiens portal, configure the following parameters:

  • PORTAL_CSS_CUSTOM: https://ambitious-coast-0a8b2110f.1.azurestaticapps.net/grid4-portal-skin-v5.0.11.css
  • PORTAL_EXTRA_JS: https://ambitious-coast-0a8b2110f.1.azurestaticapps.net/grid4-portal-skin-v5.0.11.js

CORS Configuration

These assets are configured with CORS headers to allow access from all origins (*). This ensures compatibility with your NetSapiens portals.
